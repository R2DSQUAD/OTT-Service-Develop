## 🎬 프로젝트 소개
**"어디서든 쉽게 간식처럼, SNACK"**  
다양한 엔터테인먼트 콘텐츠를 한가지 플랫폼에서 
어디서든 쉽게 간식처럼 사용자들이 간편하게 이용할 수 있는 플랫폼


## 💡 선정 배경
1. 기존 영화 예매 사이트는 수동으로 상영 일정을 등록하는 번거로움 존재  
2. 최신 영화 정보와 실시간 예매 시스템의 통합 필요

## 🔧 주요 기능
1. **OTT서비스**  
2. **영화 대여 서비스**
